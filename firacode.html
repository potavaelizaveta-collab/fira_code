ЦИФРЫ
Цифры в Fira Code выглядят как пузырьки рамунэ: ровные, аккуратные и одинакового размера. Каждый «пузырёк» сохраняет свою форму в любом начертании — от лёгкого Light до насыщенного Bold.
ЛИГАТУРЫ
В наших онигири есть «секретные ингредиенты» — лигатуры
С шрифтом Fira Code онигири склеиваются в аккуратные «начинки-лигатуры», и каждый кусочек становится красивым и удобным для чтения
